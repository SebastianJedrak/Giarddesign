GiardDesign project was made for recruitment purposes. 

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/SebastianJedrak/Giarddesign.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the dev server
   ```sh
   npm start
   ```

Ready to go!
